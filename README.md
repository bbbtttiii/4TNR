# 4TNR
4TNR: A Colorado Fourteeners guide

This CLI offers a catalog of Colorado's tallest mountains.

## Getting Started

Once all files are downloaded, run

    bundle install

To start the program, run

    ruby bin/run

## Features

Navigate to a specific mountain to view:
 - Rank
 - Range
 - Elevation
 - Prominence
 - Location (lat/long)
 - A short bio

## Built With
