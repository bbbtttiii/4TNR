# 4TNR
A Colorado Fourteeners guide

This CLI offers a catalog of Colorado's tallest mountains.

## Summary

[Getting started](#getting-started)

## Getting Started

    gem install nokogiri
    gem install httparty

## Features

Navigate to a specific mountain to view:
 - Rank
 - Range
 - Elevation
 - Prominence
 - Location (lat/long)
 - A short bio
